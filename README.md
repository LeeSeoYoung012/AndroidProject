# :smile: 카페 주문 애플리케이션 

카페에 직접가지 않고 어플로 바로 주문해서 takeout 할수 있도록 하는 카페 주문 어플을 만들었습니다. 

## :bulb: 기능
- 로그인하기 
- 회원가입
- 주문하기
- 장바구니
- 결제하기
- 포인트 충전하기 
- 위치
- 설정
- 마이페이지

## 사용기술 
- Java, 안드로이드, sqlite
